Project: Tech Store

Description:
This is a simple Python project that simulates a tech store.

The application allows:
- Adding products
- Removing products
- Searching for products
- Updating stock quantity
- Sorting products by price
- Calculating total store value
- Saving and loading products from JSON file

Files:
- models.py
- main.py
- requirements.txt
- README.txt

Libraries used:
- colorama
- tabulate

How to run:
1. Install libraries:
   pip install -r requirements.txt

2. Run the project:
   python main.py

Features:
- OOP programming
- Classes and objects
- Methods
- Loops and conditions
- File handling
- JSON storage
- Error handling
- Colored terminal output
